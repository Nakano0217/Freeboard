# Boogie-board-like drawing page

シンプルな「黒地に白文字（線）」の描画ページです。右上にツール（ペン / 消しゴム / 全削除 / 太さ調整）があり、指やマウスで描画できます。

使い方（ローカルで試す）:

1. ターミナルを開く
2. このフォルダに移動する:

```bash
cd /Users/nakanoakihiro/Desktop/名称未設定フォルダ
```

3. そのフォルダで簡易サーバを起動します（Python 3 があれば）:

```bash
python3 -m http.server 8000 --bind 127.0.0.1 --directory .
```

4. ブラウザで http://127.0.0.1:8000 を開く

仕様:
- ペンは白（初期）、消しゴムは黒で上書き（背景は黒）。
- 太さはスライダーで 1〜60px。
- 全削除はキャンバスを黒で再塗りつぶします。

必要な改善案（次のステップ）:

- undo/redo 履歴
